# 配置

## 一、京东

### 基本环境变量

- BUY_JOY_LEVEL: 购买多少级joy, 30

- JDJOY_APPLYJDBEAN: 兑换多少京豆, 2000

- JDJOY_HELPSELF: 是否循助力, true

- JD_COOKIE: 京东cokie

- GITEE_PRIVATE_KEY: gitee私钥

- GITEE_TOKEN: gitee的token

### 可选环境变量

- TG_BOT_TOKEN: tg的token

- TG_USER_ID: tg的id

- PUSH_KEY: server酱
